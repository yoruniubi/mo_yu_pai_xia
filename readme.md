## ✨ 项目简介

《摸鱼牌侠》是一款基于 **Godot 4.5** 开发的回合制卡牌游戏。玩家在战斗中通过打出 Emoji 卡牌，组成连招触发强力效果；并在 10 关流程中逐步完成核心卡进化，最终挑战 CEO 三头狮。

项目当前已形成完整可玩的主循环（战斗 → 事件 → 进化 → 终局 Boss），并支持 Web 导出与 GitHub Pages 自动部署。

---

## 🎮 核心玩法

- **Emoji 序列连击**：打出的卡牌会进入序列槽，满足配方自动触发 Combo。
- **四名可选角色**：博姆 / 墨里 / 莱奥 / 苏珊，拥有不同核心与专属连招。
- **双重进化系统**：中后期两次进化，形成流派并获得终极手段。
- **职场主题机制**：KPI、早会、虚假目标、文档塞牌、规则审计等。
- **事件关抉择**：战斗之间随机事件，影响 AP、伤害、治疗、卡池与状态。

---

## 🧩 主要特色

- 奶油色 Lofi 视觉风格 + 竖屏 4:3:3 UI 布局
- 扇形手牌、连招提示、高亮反馈、漂浮数字
- 护盾 / 闪避 / 反伤 / 易伤 / 中毒 / 火大 等状态系统
- 教学关 + 10 关主线流程 + 精英/终局 Boss 机制

## 🚀 运行方式

### 1) 本地运行（Godot）

1. 安装 **Godot 4.5.x**（建议 4.5.1）
2. 克隆仓库：

```bash
git clone https://github.com/yoruniubi/mo_yu_pai_xia.git
cd mo_yu_pai_xia
```

3. 用 Godot 打开项目目录并运行（主场景为 `Scenes/splash_screen.tscn`）

### 2) Web 版本

仓库已配置 GitHub Actions 自动导出 Web 并部署至 GitHub Pages。  
工作流文件：`.github/workflows/deploy.yaml`
